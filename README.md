# DX-learn

DX-learn is an explainability-aware AutoML framework that integrates
auto-sklearn2, SHAP-based explanations, and a TensorBoard-style
dashboard architecture.

## Features (v0.1.0)
- AutoML execution using auto-sklearn2
- SHAP-based explainability
- Explainability metrics (sparsity)
- Multi-objective model scoring
- TensorBoard-inspired dashboard backend
- FastAPI-ready architecture

## Installation

```bash
pip install dx-learn
pip install dx-learn[api]
